# BMI Calculator Web App

![BMI Calculator](https://mahapara24.github.io/BMI-app/assets/bmi.jpg)

This is a simple BMI (Body Mass Index) Calculator web application. It allows users to calculate their BMI by providing their weight (in kilograms) and height (in centimeters). The app then displays the BMI value and a corresponding BMI category, such as underweight, normal weight, overweight, or obese.

## Table of Contents

- [Features](#features)
- [Demo](#demo)
- [Installation](#installation)
- [Usage](#usage)

## Features

- Calculate BMI quickly and easily.
- Provides a BMI category based on the calculated BMI value.
- User-friendly and responsive design.
- No additional software or dependencies required.

## Demo

You can try the BMI Calculator by visiting the [live demo](https://mahapara24.github.io/BMI-app/).

## Installation

You don't need to install anything to use this web app, as it runs directly in your web browser. However, if you want to host it locally or make modifications to the code, you can clone the repository:

```bash
git clone https://github.com/mahapara24/BMI-app.git
```

## Usage

1. Open the [live demo](https://mahapara24.github.io/BMI-app/) in your web browser.
2. Enter your weight in kilograms.
3. Enter your height in centimeters.
4. Click the "Calculate BMI" button.
5. The app will display your BMI value and category.

---

Feel free to reach out if you have any questions or issues with the app. Enjoy calculating your BMI!
